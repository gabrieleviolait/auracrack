# Aura Security Suite
Aura Security Suite è un'applicazione desktop open source per svolgere controlli di sicurezza di base su password, dispositivi e traffico della propria rete. Riunisce strumenti normalmente separati in un'unica interfaccia grafica semplice, pensata per formazione, laboratori domestici, amministratori di rete e verifiche eseguite con autorizzazione
